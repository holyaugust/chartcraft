# ChartCraft - 数据可视化图表生成器

一个现代化的 Web 应用，支持手动填写表格数据或上传 Excel 文件，自动生成多种精美图表，并支持在线预览和 PNG 图片下载。

## 功能特性

- **手动填表** — 在线编辑表格，动态增删行列
- **Excel 导入** — 支持 `.xlsx`、`.xls`、`.csv` 格式，拖拽或点击上传
- **多种图表** — 柱状图、折线图、面积图、饼图、环形图、散点图、雷达图
- **实时预览** — 修改数据或切换图表类型后立即更新
- **全屏预览** — 大图模式查看图表细节
- **PNG 导出** — 高清（3x）PNG 图片下载
- **数据导出** — 将当前表格数据导出为 Excel

## 数据格式

| 分类 | 系列1 | 系列2 | ... |
|------|-------|-------|-----|
| 项目A | 100   | 200   |     |
| 项目B | 150   | 180   |     |

- 第一列：分类名称（X 轴 / 饼图标签）
- 其余列：数值数据（各数据系列）
- 散点图：前两列数值分别作为 X、Y 轴
- 饼图/环形图：使用第一列 + 第一组数值列

## 快速开始

```bash
npm install
npm run dev
```

浏览器访问 `http://localhost:5173`

## 技术栈

- React 19 + TypeScript + Vite
- ECharts 6 — 图表渲染
- SheetJS (xlsx) — Excel 解析
- Lucide React — 图标

## 构建部署

```bash
npm run build
npm run preview
```

构建产物在 `dist/` 目录，可部署到任意静态托管服务。
